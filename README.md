# 🛡️ Landing Page - Gestão AC-4 Pro

Landing page completa para divulgação do aplicativo Gestão AC-4 Pro, desenvolvido especialmente para militares, policiais e bombeiros.

## 📁 Arquivos

- `index.html` - Estrutura principal da página
- `styles.css` - Estilos CSS responsivos e modernos
- `script.js` - Interatividade (FAQ, formulário, animações)

## 🎨 Design

- **Tema**: Dark mode com destaque verde militar
- **Público-alvo**: PMs, bombeiros, forças de segurança pública
- **Diferencial destacado**: Cálculo automático de AC-4
- **Responsivo**: Desktop, tablet e mobile

## 💳 Como Configurar Pagamentos (Brasil)

### Opção 1: Mercado Pago (Recomendado)

**Vantagens:**
- ✓ Aceita Pix, boleto, cartão
- ✓ Checkout transparente
- ✓ Taxas competitivas

**Passos:**
1. Crie conta em [mercadopago.com.br](https://www.mercadopago.com.br)
2. Ative "Checkout Pro" ou "Checkout API"
3. Integre no seu app/backend (precisa de desenvolvimento)
4. Gere links de pagamento para a landing page

**Taxas:**
- Pix: ~0,99%
- Cartão: ~4,49%
- Boleto: ~3,19%

### Opção 2: Stripe

**Vantagens:**
- ✓ Checkout profissional
- ✓ Aceita cartões internacionais
- ✓ Recorrência automática

**Desvantagens:**
- Pix/boleto exige integração adicional
- Taxas em dólar

**Taxas:**
- Cartão nacional: 3,6% + R$ 0,40
- Cartão internacional: 4,5% + R$ 0,40

### Opção 3: PagSeguro

**Vantagens:**
- ✓ Conhecido no Brasil
- ✓ Muitas opções de pagamento

**Desvantagens:**
- Interface menos moderna
- Taxas mais altas

**Taxas:**
- Pix: 0,99%
- Cartão: 3,99% + R$ 0,10
- Boleto: 3,49% + R$ 0,10

### Opção 4: Asaas (Recomendado para SaaS)

**Vantagens:**
- ✓ Cobrança recorrente nativa
- ✓ API completa
- ✓ Taxas baixas

**Passos:**
1. Cadastre-se em [asaas.com](https://asaas.com)
2. Configure sua conta bancária
3. Use a API para gerar cobranças
4. Link de pagamento por email

**Taxas:**
- Pix: 0%
- Cartão: 2,49%
- Boleto: R$ 2,49 por boleto

### 💡 Recomendação para o Gestão AC-4 Pro:

**Para começar (MVP):**
1. Use **Asaas** ou **Mercado Pago**
2. Gere links de pagamento manualmente
3. Envie por email após o teste gratuito de 30 dias

**Para escalar:**
1. Integre a API no seu backend Firebase
2. Automatize a geração de cobranças
3. Use webhooks para ativar/desativar acesso
